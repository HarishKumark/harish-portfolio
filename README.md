# Harish Portfolio

A responsive, dependency-free portfolio built with HTML, CSS and JavaScript. It is ready to deploy on Vercel as a static site.

## Before you publish

Open `index.html` and replace these placeholders:

1. `harish.email@example.com` — your real email address.
2. The two `href="#"` social links — your LinkedIn and GitHub URLs.
3. `assets/harish-cv.pdf` — add your CV to an `assets` folder using this exact filename, or change the link to your preferred CV file.
4. Review the experience dates and content for accuracy.

## Run it locally

No install is needed. Open `index.html` directly in a browser, or use a local server:

```bash
npx serve .
```

## Deploy with Vercel using GitHub

1. Create a new GitHub repository, for example `harish-portfolio`.
2. Upload these files and push them to the `main` branch.
3. In Vercel, select **Add New → Project** and import the GitHub repository.
4. Leave the framework preset as **Other**. No build command is required.
5. Click **Deploy**.

Vercel can deploy files as-is when no framework is detected, and connecting a Git repository enables automatic deployments when you push updates.

## Deploy from the command line

```bash
npm i -g vercel
vercel
```

Run the command from this project folder and follow the prompts.
